# MVTEC E-commerce — Guadeloupe / Métropole

Site e-commerce MVTEC avec deux fronts publics et un back office commun.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- React Router
- Lucide React

## Installation

```bash
npm install
npm run dev
```

## Routes

- `/guadeloupe` : storefront local Guadeloupe
- `/metropole` : storefront France métropolitaine
- `/boutique` : catalogue
- `/location` : location
- `/sav` : maintenance / SAV
- `/contact` : contact / devis
- `/admin` : back office MVTEC

## Commandes qualité

```bash
npm run typecheck
npm run build
```
